正式数据可获取 2014 年 4 月 30 日至今的个股 ESG 评价数据，日度更新。

米筐 API 提供的秩鼎 ESG 数据一共分为三个横向层级 level（level 0、level 1 和 level 2）和三个纵向层级 type（E，S，G）。

请先单独安装 rqdatac_esg，导入后使用

Level 0：ESG 综合评价等级和得分

ESG 综合评价等级：根据 ESG 综合评价分数高低分为高（AAA、AA、A）、中（BBB、BB、B）、低（CCC、CC、C）3 档共 9 个等级。

ESG 综合评价得分：取值范围为 0-100 分，根据环境、社会责任、治理 3 个一级维度综合计算得出。

Level 1： 包含环境，社会责任，治理这三个方面的评价等级和得分

得分：取值范围为 0-100 分，评价得分根据单个一级维度下的二级维度评价得分综合计算得出。（针对不同行业采用不同权重计算）

Level 2：4+5+5 共 14 个二级维度的评价等级和得分

得分：取值范围为 0-100 分，评价得分根据单个二级维度下的细分指标综合计算得出。

随着注册制的正式落地，A 股股票数量已经超过 5000 只，其中小市值股票也在这几年引起了市场的广泛关注。米筐小市值概念指数旨在为投资者提供反映 A 股市场小市值风格特征的代表指数，既可作为被动投资标的，也可作为量化投资策略的业绩比较基准。用户可通过米筐 RQData 金融数据 API 获取指数历史及实时行情数据，亦可在米筐 RQAMS 资产管理系统中选择米筐小市值指数作为业绩基准，对风格匹配的产品或投资组合进行深度分析。

米筐小市值概念指数与米筐小市值概念全收益指数采用“等权”的编制方法，每日平衡，即每一只成分股每日对于指数收益率的影响是等权的。不同的是，米筐小市值概念指数为价格指数，不对成分股分红进行调整，任由其价格自行回落，米筐小市值概念全收益指数则考虑到成分股的权益类事件以进行调整，例如对股票分红、拆分、送股等权益类事件进行复权。

米筐可转债等权指数是可用于衡量可转债市场整体表现的指标，其变动可以反映市场对可转债的需求和情绪，对投资者进行投资决策提供了重要的参考依据。在投资组合中，米筐可转债等权指数也可以作为衡量可转债类投资表现的基准。通过关注米筐可转债等权指数，投资者可以更好地把握市场动态，制定更加有效的投资策略。

米筐微盘股指数是由 A 股中市值居于后 400 的极小市值的个股等权构建而成。通过微盘股指数，投资者可以更好地把握微小市值股票市场的走势，结合自身策略，获得更好收益。

双创板聚焦创新型、创业型优质企业，集中体现了新兴产业的成长潜力与发展方向，是 A 股市场中兼具创新性与成长性的重要板块。通过双创板指数，助力投资者更清晰地把握双创领域的市场机遇与投资脉络。双创指数包含了创业板和科创板的所有股票，为了分析和使用方便，我们编制了以下两种加权的指数。另外由于科创板在2019年7月22日才开始正式运行，因此双创指数从该日开始，后续保持每日增量更新。

波动性指数（VIX）作为一个实时市场指标，反映了市场对未来 30 日内波动性的预期。在投资决策过程中，投资者常利用 VIX 指数来评估市场风险程度、恐慌情绪或压力状况。该指数所用期权到期月份的选取规则如下：若查询日期早于当月期权到期日，则以查询日所在月份为期权到期月；若查询日期晚于当月期权到期日，则以次月为期权到期月。

其中，NT1 表示当前时间距离当期合约结束的分钟数（本月到期），NT2 表示当前时间距离下期合约结束的分钟数（次月到期），NT30 表示 30 天的分钟数，NT365 表示本年分钟数，σ12 表示当期波动率，σ22 表示下期波动率。

全A指数覆盖A股全市场样本，旨在反映A股市场整体走势。采用自由流通市值加权方法，以全收益方式进行计算。 用户可通过米筐 RQData 金融数据 API 获取指数历史及实时行情数据。

全A等权指数覆盖A股全市场样本，旨在反映A股市场整体走势。采用等权重方法，以全收益方式进行计算。 用户可通过米筐 RQData 金融数据 API 获取指数历史及实时行情数据。

本指数覆盖沪深交易所全部A股，剔除ST股票以排除异常波动干扰，更纯粹反映市场整体走势。采用等权重全收益计算。 用户可通过米筐 RQData 金融数据 API 获取指数历史及实时行情数据。
