# Seedance 提示词使用技巧

## 提示词参数传入方式

**新方式（推荐）** - 在 request body 中直接传入参数（强校验）：

```json
{
    "model": "doubao-seedance-1-5-pro-251215",
    "content": [{"type": "text", "text": "小猫对着镜头打哈欠"}],
    "resolution": "720p",
    "ratio": "16:9",
    "duration": 5,
    "seed": 11,
    "camera_fixed": false,
    "watermark": true
}
```

**旧方式** - 在提示词后追加 `--[parameters]`（弱校验）：

```
小猫对着镜头打哈欠 --rs 720p --rt 16:9 --dur 5 --seed 11 --cf false --wm true
```

## 图生视频提示词技巧

### 基础结构

```
提示词 = 主体 + 运动, 背景 + 运动, 镜头 + 运动
```

### 关键原则

1. **减少静止描述**：图生视频已有场景，多描述运动部分（主体运动、背景运动/变化、镜头运动）
2. **简单直接**：使用简单词语和句子结构，模型会自动扩写
3. **特征描述**：主体有突出特征时加上特征定位（如"老人"、"戴墨镜的女人"）。运动描述时关键程度副词要明确（如"快速"、"幅度大"）
4. **遵从图片**：基于输入图片内容来写，不要与图片内容/基础参数存在事实矛盾
5. **负向提示词不生效**：模型不响应负向提示词（如"不要..."）

### 示例

```
老人戴上眼镜 | 固定镜头，720p，10s
打太极，镜头环绕，聚焦脸部 | 不固定镜头，720p，5s
```

## 多个连续动作提示词

模型支持时序性的多个连续动作，以及多个主体的不同动作：

```
提示词 = 主体1 + 运动1 + 运动2
提示词 = 主体1 + 运动1 + 主体2 + 运动2
```

**示例：**
- `转过脸对着镜头向前走，然后停下，脸上露出愤怒的表情，然后叉腰`
- `镜头聚焦于背景的老师，前景的女孩变得模糊，老师非常生气的骂人`
- `女人一边哭泣一边喝酒，一个男人走进来安慰她`

## 运镜提示词

> **注意**：有运镜提示词时，基础参数要选择「不固定镜头」（`camera_fixed: false`）

### 镜头切换
使用"镜头切换"关键词连接不同镜头：
- `小猫和小狗吃猫粮，镜头切换到特写猫粮颗颗分明`
- `全景拍摄，模特微笑着走近镜头。镜头切换，近景拍摄模特下半身`

### 平移
镜头向上/下/左/右移动：
- `镜头缓慢向下移动，到街道，路灯`

### 变焦
镜头拉远/推近/聚焦/特写：
- `镜头特写到帐篷，360度展示帐篷全景`
- `特写镜头，然后镜头拉远，看到是一个小孩雕塑家`

### 环绕
镜头环绕 360 度展示：
- `镜头环绕运镜`
- `环绕镜头，缓慢推进至人物的脸部特写`

### 航拍
航拍/广角镜头：
- `男人张开双臂，航拍环绕镜头`

### 手持
模拟手持镜头效果：
- `手持镜头，画面微微抖动体现手持感`

### 跟随
镜头跟随主体：
- `镜头跟随一位驾驶哈雷摩托的男性`

## 程度副词

突出动作频率与强度。模型无法从参考图获取运动程度，需在 prompt 中明确。可适当夸大程度增强表现力。

**关键词：** 快速、剧烈、大幅度、高频率、强力、疯狂 ...

**示例：**
- `车辆快速往前开，穿过时空通道，光影流动，镜头快速跟随`
- `双腿快速交替，双臂有力摆动，在赛场上全力冲刺`

## 文生视频提示词技巧

### 基础结构

```
提示词 = 主体 + 运动 + 场景 + 镜头、风格...
```

主体 + 运动 + 场景是最核心基本要素。图生视频的连续动作、运镜、程度副词技巧同样适用。

### 详细描述技巧

- **人物描写**：外表、着装、姿势
- **环境细节**：自然环境或建筑环境
- **情绪与动态结合**：人物情绪状态和环境动态
- **氛围渲染**：光线描述（黄昏、清晨、昏暗、温暖的光等）

### 风格关键词

支持的风格：`国漫`、`水墨`、`水彩`、`日漫`、`美漫`、`剪纸`、`体素`、`毛毡`、`线稿`

## 有声视频提示词技巧 (Seedance 1.5 pro)

- 模型基于文本提示词与视觉内容，自动生成匹配的人声、音效及背景音乐
- **建议将对话部分置于双引号内**，以优化音频生成效果

**示例：**
```
男人叫住女人说："你记住，以后不可以用手指指月亮。"
```

## 常见问题与解决

### 问题 1：视频肢体崩坏
**解决**：多试几次（换 seed），或通过 prompt 避开展示手/脚部的镜头

### 问题 2：图生视频没有遵循重要的程度副词
**解决**：删除不必要的内容（已在图中体现的），强调重要的程度副词

**示例**：将冗长描述简化为 `一只龙快速的拍动翅膀，翅膀运动的幅度很大。仰拍，镜头旋转绕行。压迫感`

### 问题 3：图生视频结果某些地方不符合要求
**解决**：在 prompt 中重新强调问题细节

**示例**：添加 `他眼神自然` 来修正眼神问题
